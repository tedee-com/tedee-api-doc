Auto unlock pull condition
---------------------------------------

This enum describes possible conditions for pull spring in auto unlock

+-----------+-----------------------+
| Number    | Name                  |
+===========+=======================+
| 1         | AutoPullBased         |
+-----------+-----------------------+
| 2         | Always                |
+-----------+-----------------------+
| 3         | Never                 |
+-----------+-----------------------+
| 4         | Locked                |
+-----------+-----------------------+
| 5         | Unlocked              |
+-----------+-----------------------+
