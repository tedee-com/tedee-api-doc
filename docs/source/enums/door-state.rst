Door state
---------------------------------------

This enum describes possible states of door

+-----------+-----------------------+
| Number    | Name                  |
+===========+=======================+
| 0         | NotPaired             |
+-----------+-----------------------+
| 1         | Disconnected          |
+-----------+-----------------------+
| 2         | Opened                |
+-----------+-----------------------+
| 3         | Closed                |
+-----------+-----------------------+
| 4         | Uncalibrated          |
+-----------+-----------------------+

.. note::
    To get values different than 0, door sensor must be paired with lock.
