Open parameter
----------------

This enum describes possible open parameters for unlocking lock

+-----------+---------------------------------------+
| Number    | Name                                  |
+===========+=======================================+
| 0         | Default behaviour of unlocking lock   |
+-----------+---------------------------------------+
| 1         | Used for auto unlocking               |
+-----------+---------------------------------------+
| 2         | Force unlock                          |
+-----------+---------------------------------------+
