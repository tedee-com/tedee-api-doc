Access level
---------------------------------------

This enum describes user access level to the device

+-----------+-----------------------+
| Number    | Name                  |
+===========+=======================+
| 0         | Guest                 |
+-----------+-----------------------+
| 1         | Admin                 |
+-----------+-----------------------+
| 2         | Owner                 |
+-----------+-----------------------+
| 3         | None                  |
+-----------+-----------------------+
