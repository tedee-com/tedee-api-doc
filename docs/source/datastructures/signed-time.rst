Signed time
-----------------

+-----------+--------+------------------------------+
| Name      | Type   | Description                  |
+===========+========+==============================+
| datetime  | string | signed time as base64 string |
+-----------+--------+------------------------------+
| signature | string | signature of signed time     |
+-----------+--------+------------------------------+

.. note::
    Signed time and signature can be verified by Tedee device only.
