Lock properties
-----------------

+------------------------+------------+--------------------------------------------------------------+
| Name                   | Type       | Description                                                  |
+========================+============+==============================================================+
| state                  | number     | represents current state of the lock                         |
+------------------------+------------+--------------------------------------------------------------+
| isCharging             | boolean    | represents if lock is currently charging                     |
+------------------------+------------+--------------------------------------------------------------+
| batteryLevel           | number     | represents current battery level                             |
+------------------------+------------+--------------------------------------------------------------+
