Organization users
==================

Endpoints used for managing users within an organization.

.. toctree::
   :maxdepth: 1
   :hidden:

   Get all <get-all>
   Invite user <create>
   Get profile <get-profile>
   Update profile <update>
   Assign admin role <assign-admin>
   Remove admin role <remove-admin>
   Remove user <delete>

**Operations**

- `Get all <get-all.html>`_ - List organization users with optional filters
- `Invite user <create.html>`_ - Add a new user to the organization
- `Get profile <get-profile.html>`_ - Get details of a single organization user
- `Update profile <update.html>`_ - Update organization user display name
- `Assign admin role <assign-admin.html>`_ - Grant admin role to a user
- `Remove admin role <remove-admin.html>`_ - Revoke admin role from a user
- `Remove user <delete.html>`_ - Delete a user from organization
